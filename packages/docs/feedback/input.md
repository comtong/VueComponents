### TInput

#### 使用指南

```JavaScript
import { TInput } from '@lw33/tui'
```

#### 属性


| 参数        | 说明                            | 类型     | 默认值                         |
| ----------- | ------------------------------ | -------- | ------------------------------ |
| value       | 当前input值，可用v-model双向绑定 | String   |                                |
| textAlign   | 对齐方式                        | String   |  left                         |
| disabled    | 是否禁用                        | String   |                                |
| newClass  | 自定义样式                 | boolean  | false                          |
