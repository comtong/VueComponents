### TCell

#### 使用指南

```JavaScript
import { TCell } from '@lw33/tui'
```

#### 属性


| 参数        | 说明                      | 类型     | 默认值                         |
| ----------- | ------------------------ | -------- | ------------------------------ |
| label       | 左边标题文字              |  String   | ""                           |
| value       | 右侧文字（复杂用slot）     | String   | ""                           |
| subTitle   | 副标题（标题下面的描述文字）| String   | ""                          |
| link        | 点击跳转链接              | String   | ""                          |
| isLink     | 是否是链接（是否展示右侧小箭头）| boolean   | 默认 false              |

#### 插槽


| 名称        | 说明                          | 
| ----------- | --------------------------   |
| value       | 右侧内容                      |
| icon        | 标题左侧图像位置              | 
| title       | 标题                          | 
| sub-title   | 副标题（标题下面的描述文字）    |
