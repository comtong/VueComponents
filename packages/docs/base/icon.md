### TIcon

#### 使用指南

```JavaScript
import { TIcon } from '@lw33/tui'
```

#### 属性


| 参数        | 说明               | 类型     | 默认值                         |
| ----------- | ------------------ | -------- | ------------------------------ |
| type        | 按钮类型          | String   | 可选类型为'down','up','back','loading','close','radioCheck','radioUncheck'|
| size        | 大小              | String   | 默认 1rem                           |
