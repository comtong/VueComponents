### TCellInput

#### 使用指南

```JavaScript
import { TCellInput } from '@lw33/tui'
```

#### 属性


| 参数        | 说明                            | 类型     | 默认值                         |
| ----------- | ------------------------------ | -------- | ------------------------------ |
| value       | 当前input值，可用v-model双向绑定 | String   |                                |
| textAlign   | 对齐方式                        | String   |  left                         |
| label       | 文本框前面的标签                 | String   |                                |
| disabled    | 是否禁用                        | String   |                                |
| showClear  | 是否展示清空按钮                 | boolean  | false                          |
